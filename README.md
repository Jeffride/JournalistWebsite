# JournalistWebsite
Commissioned website for journalist Dawn Attride.

# URL: http://dattride.com/

This is a portfolio website which will comprise of the following sections:
- about me
- articles
- gallery
- projects

## Prereqs

- python 3.7 or later
- pip
- virtualenv
- awsebcli

## Starting the venv
